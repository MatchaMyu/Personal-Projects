# JavaTestNotePad

A simple test running java without using an environment. Uses notepad++ and instructions on how to run a java program without an environment.

# How to run:

Create the program and save it as a .java extension.
Then, open the command/root and navigate to the folder the file is in.
Run "javac NAMEOFFILE.java"
This will create a second file if coded properly, as a class.

Run "java NAMEOFFILE" and the program to execute.
